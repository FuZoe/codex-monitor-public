# Third-Party Notices

This project does not currently bundle any third-party animation assets.

The status character displayed on the Android app is an original SVG + CSS
animation embedded in `android/assets/status-character.html`. It does not
use external images or libraries.

### Historical Note

Earlier versions of this project included Clawd animation GIFs sourced from
[clawd-on-desk](https://github.com/rullerzhou-afk/clawd-on-desk) under the
MIT License. These assets were removed and replaced with an original SVG
robot character to eliminate third-party IP/licensing dependencies.

### Codex Event Mapping

The JSONL event-to-state mapping logic in `desktop/codex-monitor-server.py`
was informed by the Codex-related state mapping in `clawd-on-desk`.
